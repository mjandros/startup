## HTML Basics

Anchor (URL):
```
<a href = "https://linktosite.com">DisplayText</a>
```
Image:
```
<img src = "path/to/img.png" alt = "Alt Text"/>
<img src = "https://linktoimage.com" alt = "Alt Text"/>
```
List:
```
<p>List Title</p>
<ul>
    <li>Element 1></li>
    <li>Element 2></li>
</ul>
```
Table:
```
<table>
    <tr>
        <th>Upper Left</th>
        <th>Upper Right</th>
    </tr>
    <tr>
        <th>Lower Left</th>
        <th>Lower Right</th>
    </tr>
<table>
```
Other:
```
<h1>Title Text</h1>
<header> ... </header>
<body> ... </body>
<section> ... </section>
<p>text</p>
<main> ... </main>
<aside> ... </aside>
<footer> ... </footer
```
